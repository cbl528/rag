# AI 智能客服产品介绍

> 产品名称：智答（SmartReply）
> 版本：v2.5  
> 发布日期：2026 年 7 月

## 一、产品简介

智答是一款基于大语言模型的企业级智能客服平台，融合 RAG（检索增强生成）、多轮对话管理和人机协作机制，帮助企业快速搭建高效、精准的客户服务系统。

## 二、核心特性

### 2.1 智能问答引擎

基于先进的 LLM 技术，结合企业专属知识库，实现高准确率的意图识别和答案生成：

- 支持 **100+** 业务场景的意图识别
- 基于向量检索的实时知识库匹配
- 多轮对话上下文记忆，支持 20 轮连续对话
- 置信度低于阈值时自动转人工

### 2.2 知识库管理

- 支持批量导入 FAQ、产品文档、操作手册（支持 TXT、MD、PDF、DOCX 格式）
- 自动文档分片与向量化索引
- 知识库版本管理与 A/B 测试
- 支持结构化知识图谱

### 2.3 全渠道接入

对接主流通讯渠道：

| 渠道 | 支持版本 | 接入方式 |
|------|----------|----------|
| 微信 | 公众号 / 小程序 | SDK / API |
| 企业微信 | 所有版本 | Webhook |
| 钉钉 | 所有版本 | Stream 模式 |
| 飞书 | 所有版本 | Bot API |
| Web | 桌面 / 移动端 | WebSocket / SSE |
| 电话 | IVR 集成 | SIP 对接 |

### 2.4 人工协作

提供完整的坐席工作台：

1. **智能辅助** — 实时为坐席推荐回复话术和解决方案
2. **对话摘要** — 自动生成客服对话摘要，减少事后填报
3. **工单系统** — 支持一键生成工单，跟追踪进度
4. **质量监控** — 支持对话质检评分和数据分析报表

### 2.5 数据分析与监控

- 实时运营看板：会话量、满意度、解决率等核心指标
- 热点话题分析：自动挖掘高频问题和趋势变化
- 知识库效果评估：命中率、答案采纳率、转人工率等
- 全链路调用监控：LLM 调用延迟、Token 消耗、向量检索耗时

## 三、技术栈

| 组件 | 技术选型 |
|------|----------|
| LLM | DeepSeek / GLM-4 / Qwen2（可切换） |
| 向量数据库 | Milvus |
| 文档处理 | FixedSizeBoundaryChunker |
| 后端框架 | Spring Boot 3.x + JDK 17 |
| 前端框架 | React 19 + Tailwind CSS |
| 实时通信 | SSE（Server-Sent Events） |
| 认证授权 | Sa-Token + RBAC |

## 四、部署指南

### 4.1 环境要求

```
JDK:    17+
内存:   16GB+
存储:   100GB+
CPU:    4核+
操作系统: Linux / macOS / Windows
数据库: MySQL 8.0+
向量库: Milvus 2.4+
```

### 4.2 快速启动

```bash
# 1. 克隆项目
git clone https://github.com/example/smartreply.git

# 2. 启动数据库
docker compose up -d mysql milvus

# 3. 配置 API Key
cp application-dev.yaml.example application-dev.yaml
# 编辑 application-dev.yaml 填写 API Key

# 4. 启动服务
mvn spring-boot:run -pl business -am

# 5. 启动前端
cd fronted && npm install && npm run dev
```

### 4.3 验证部署

访问 `http://localhost:5173`，使用管理员账号登录后：

1. 进入 **管理控制台** → **文档上传**
2. 上传一份产品文档（.txt 或 .md）
3. 返回聊天页面，向机器人提问文档中的内容
4. 确认回答是否准确引用了上传的知识
